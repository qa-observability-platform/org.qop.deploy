Features

Everything your QA team needs

One platform to monitor all your tests — regardless of the framework.

⚡

Real-time Execution Streaming

Watch tests run live via WebSocket. See pass/fail status as each test completes.

🔍

Flaky Test Detection

Automatic flakiness scoring (0–100), pattern recognition, and root cause analysis for unstable tests.

🤖

AI-Powered Analysis

Failure analysis powered by Claude, GPT, or Groq. Get human-readable explanations for every failure.

Self-hosted Setup

Up and running in minutes

No cloud account required. Runs entirely on your own infrastructure.

1

Clone the deploy repo

Get the docker-compose configuration from GitHub.

2

Configure environment

Copy .env.docker.example to .env.docker and set your secrets.

3

Start with Docker Compose

All 4 services start automatically — database, API, AI backend, and dashboard.

4

Install the reporter

Add the QOP reporter to your test project and point it at your server.

# 1. Clone
git clone https://github.com/qa-observability-platform/org.qop.deploy.git && cd org.qop.deploy

# 2. Configure
cp .env.docker.example .env.docker

# 3. Start
docker compose --env-file .env.docker up -d

# 4. Install reporter (example: Playwright)
npm install @qa-observability-platform/playwright

Start monitoring your tests today

Free, open-source, and fully self-hosted. Your data stays on your server.
